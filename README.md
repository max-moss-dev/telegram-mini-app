# Telegram Mini App

Простий Telegram Mini App, розгорнутий на GitHub Pages.

## URL для Telegram Bot

Після активації GitHub Pages, твій додаток буде доступний за адресою:
```
https://max-moss-dev.github.io/telegram-mini-app
```

## Налаштування Web App в BotFather

1. Напиши `@BotFather` → `/mybots`
2. Обери свого бота → **Bot Settings** → **Menu Button** → **Configure menu button**
3. Або **Back to Bot** → **Inline mode** → **Configure Web App**
4. Введи URL: `https://max-moss-dev.github.io/telegram-mini-app`
